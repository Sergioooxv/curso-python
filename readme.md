## ¿Qué es este repositorio?

Repositorio personal donde documento mi aprendizaje de Python desde cero hasta nivel avanzado.
Está pensado para que cualquier persona pueda seguirlo en orden y aprender Python de forma progresiva.

Cada carpeta corresponde a una categoría del lenguaje y contiene:

- 📝 Comentarios explicando cada concepto
- 💻 5 ejercicios de menor a mayor dificultad
- 💬 Código comentado línea a línea

---

## 🛠️ Requisitos

```bash
python --version  # Python 3.10 o superior
```

Descarga Python en [python.org/downloads](https://python.org/downloads)  
Editor recomendado: [Visual Studio Code](https://code.visualstudio.com)

---

## ▶️ Cómo ejecutar un ejercicio

```bash
git clone https://github.com/Sergioooxv/curso-python.git
cd curso-python
python 01_basicos/01_variables/1_ejercicio.py
```

## 💡 Consejos

- Haz los ejercicios tú solo antes de buscar la solución
- Lee los comentarios del código — explican cada línea
- No copies y pegues — escríbelo tú aunque ya lo hayas visto
- Si algo no funciona es parte del aprendizaje — lee el error e investiga
- Avanza en orden — cada categoría depende de las anteriores
